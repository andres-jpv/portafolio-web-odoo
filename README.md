# 🌐 Portafolio Web — Construido con Odoo v19

Portafolio personal desarrollado íntegramente con el framework **Odoo v19**, demostrando el uso de Odoo como plataforma de desarrollo web más allá de su uso empresarial tradicional.

---

## 🎯 Objetivo

Mostrar las capacidades de Odoo como framework web para construir sitios personalizados, combinando:
- Módulos de sitio web de Odoo (Website Builder)
- Plantillas QWeb personalizadas
- Estilos CSS / Bootstrap integrados en el ecosistema Odoo

---

## 🛠️ Stack

| Tecnología | Uso |
|---|---|
| Odoo 19.0 | Framework base (Website module) |
| QWeb | Plantillas HTML del sitio |
| CSS / Bootstrap | Estilos y diseño responsive |
| Docker / DevContainers | Entorno de desarrollo |
